How to enable cloud sync
  1. Go to console.firebase.google.com and create a free project (Spark plan)
  2. Enable Google Sign-In under Authentication → Sign-in method → Google
  3. Create a Firestore Database → Start in test mode
  4. Open js/auth.js and fill in your config under FIREBASE_CONFIG
  5. Reload the page — the "Sign in to sync" button will now work
